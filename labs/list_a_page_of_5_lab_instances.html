Labs API

Create and retrieve state information on lab_instances

List a page of 5 lab_instances

GET /api/v1/labs/:id/lab_instances

Parameters

Name Description
page Which pagination page (default 1)
per_page How many items per page (default 30)
sort_by Sort by choice. Options are id, status, lab_id, build_seconds, alive_minutes, progress, errored_at, created_at, updated_at, finished_at, and destroyed_at
direction Sort direction. Options are asc or desc
status Filter by status. 1=Starting, 2=Running, 3=Shutdown, 4=Error
lab_id Filter by parent lab id
created_before Filter by lab instances created before this time
created_after Filter by lab instances created after this time
search Filter by matching UUID, will return partial matches
