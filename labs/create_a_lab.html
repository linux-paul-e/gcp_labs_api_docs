Labs API

Create and retrieve state information on labs

Create a lab

POST api/v1/labs

Parameters

Name Description
name The name of the lab
api_name The api name of the lab
configuration The yaml configuration file as a string
max_run_minutes The maximum time in minutes permitted for this lab
