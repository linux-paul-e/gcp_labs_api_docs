Labs API

Create and retrieve state information on labs

List a page of 5 labs

GET /api/v1/labs

Parameters

Name Description
page Which pagination page (default 1)
per_page How many items per page (default 30)
sort_by Sort by choice. Options are id, name, api_name, created_at, updated_at, and max_run_minutes
direction Sort direction. Options are asc or desc
created_before Filter by lab instances created before this time
created_after Filter by lab instances created after this time
search Filter by matching name or api_name, will return partial matches.

Request

Headers

Accept: application/json
Content-Type: application/json
Host: example.org
Cookie:

Route

GET /api/v1/labs?page=1&per_page=5&sort_by=name&direction=desc

Query Parameters

page: 1
per_page: 5
sort_by: name
direction: desc
