Labs API

Create and retrieve state information on accounts

List a page of 5 accounts

GET /api/v1/accounts

Parameters

Name Description
page Which pagination page (default 1)
per_page How many items per page (default 30)
sort_by Sort by choice. Options are id, email, in_use, in_rotation, last_used, created_at, and updated_at
direction Sort direction. Options are asc or desc

Request

Headers

Accept: application/json
Content-Type: application/json
Host: example.org
Cookie:

Route

GET /api/v1/accounts?page=1&per_page=5&sort_by=email&direction=asc

page: 1
per_page: 5
sort_by: email
direction: asc
